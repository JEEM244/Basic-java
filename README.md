# Ejemplo: ¿Cómo funciona Java?

Este pequeño ejemplo muestra conceptos básicos de Java: variables, arreglos, métodos, clases, bucles y estructuras de control.

## Archivos
- `main.java` — programa de ejemplo que puedes compilar y ejecutar.

## Compilar y ejecutar (Windows PowerShell)

```powershell
javac main.java
java Main
```

Notas:
- Asegúrate de tener `javac` en el PATH (JDK instalado).
- `javac` compila el archivo y crea archivos `.class`; `java Main` ejecuta la clase con el método `main`.
